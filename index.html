Bridge Race

Description:

A 3D Casual Game where you control a character who runs and collects blocks to build staircase and reach the end of the stage.
Move quickly to collect blocks that match your color and place them where a staircase is needed to move to the next level.
If an opponent starts building a staircase at the same spot as you, the blocks will be painted in their color, so you will have to use new blocks to repaint them.

Knowlege:
  • Physics 3D, Raycast, NavMesh, State Machine, Scriptable Object, Asset Joystick
  • Design pattern: Singleton, Observer, Object Pooling
  • OOP: Inheritance, Polymorphism
Github + Demo

Move Stop Move

Description:

A Casual Game where you control a 3D stickman and move freely in a map.
Move your character by sliding your finger over your smartphone's screen in whichever direction you prefer.
You will have to fight against many enemies and kill them by using different weapons. The more enemies you killed, the bigger you will become.

Knowlege:
  • Physics 3D, NavMesh, State Machine, Scriptable Object, Resources.Load
    Component Cache, Json Data, Asset Joystick, Asset Offscreen Target Indicator
  • Design pattern: Singleton, Observer, Object Pooling
  • OOP: Inheritance, Polymorphism
